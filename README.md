## 📌 Project Overview

**DeshiSpices BD** is a front‑end brand concept website that I built as a **self‑initiated practice project** during my **2nd Year, 1st Semester** of CSE at **Metropolitan University**. This project was **not a course assignment** – it was my personal effort to explore advanced front‑end web technologies, creative UI/UX patterns, and modern CSS/JavaScript techniques outside the classroom.

The website showcases a fictional Bangladeshi spice brand with a fully responsive product showcase, interactive animations, and a premium visual identity. My goal was to push my skills beyond the syllabus and build something portfolio‑ready.

## 🎯 Purpose & Learning Goals

- Experiment with **glassmorphism**, **custom cursors**, and **CSS‑only animated backgrounds**.
- Implement a **fully responsive layout** with mobile navigation (hamburger menu).
- Practice **dynamic content filtering** (product categories) using vanilla JavaScript.
- Build a **rich user interface** with hover effects, marquees, and scroll‑triggered styling.
- Learn to structure a **complete brand concept** from scratch – including visual hierarchy, color palette, and typography.

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| **Custom Cursor System** | A golden dot + ring that follows mouse movement, expands on interactive elements. |
| **Auto‑scrolling Marquee** | Top bar with rotating messages (“Authentic Bangladeshi Spices …”) – infinite scroll. |
| **Sticky Glassmorphism Navbar** | Fixed navigation that becomes blurred and semi‑transparent on scroll. |
| **Animated Hero Section** | 7 background images cross‑fading in a continuous cycle + floating CSS particle system. |
| **Statistics Counter Bar** | Four stat cards (Years of Excellence, Spice Varieties, Happy Customers, Awards) with animated numbers. |
| **Filterable Product Grid** | Products displayed in responsive cards; filterable by category (All, Whole Spices, Ground Spices, Blends). |
| **Product Card Interactions** | Hover scaling, border glow, overlay buttons (“Quick View”, “Add to Cart”). |
| **Image Gallery Marquee** | Horizontal scrolling strip of spice product images (autoplay). |
| **Mobile‑friendly Navigation** | Hamburger menu that collapses the nav links on small screens. |
| **Smooth Scrolling & Active Link Highlight** | Section navigation with smooth scroll and visual active state. |
| **Chili Logo Animation** | The brand logo (chili emoji) rotates on hover, adding playful branding. |

## 🧠 What I Learned

- **Advanced CSS animations** – creating a pure CSS cross‑fading background slideshow with staggered keyframes.
- **Building a custom cursor** – overriding the default cursor and synchronising two divs with mouse movement.
- **Glassmorphism in practice** – balancing transparency, blur, and borders to achieve a modern UI.
- **Filterable content** – using JavaScript to show/hide product cards based on category tabs.
- **Responsive navigation** – implementing a hamburger menu that toggles the nav links on mobile.
- **Performance considerations** – using CSS animations instead of JavaScript for the background cycle to reduce CPU load.
- **Brand design** – choosing a coherent colour palette (gold, dark tones, cream) and typography to convey premium authenticity.
